Все скрипты находятся в директории src. Вы можете сгенерировать html-страницу с названиями,
описаниями и ссылками на скрипты. Для этого:

1)	В Windows: просто запустите файл createIndex.exe
	В Linux:
		- сделай файл createIndex исполняемым
			$ chmod 755 createIndex
		- запустите его на исполнение:
			$ ./createIndex

2) Откройте созданный index.html в браузере.

P.S.
Исходные коды генераторов html-страницы находятся в директории .src
